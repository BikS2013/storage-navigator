# Suggested commands
- `npm run build` — compile TypeScript to `dist`.
- `npm run cli -- <args>` — run the CLI entrypoint via `tsx`.
- `npm run ui` — launch the Electron app entrypoint.
- `git status`, `git diff`, `git log --oneline --decorate --graph --max-count=10` — basic repo inspection.
- `rg <pattern>` / `rg --files` — fast search on Darwin.
